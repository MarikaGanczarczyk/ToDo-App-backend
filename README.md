✅ To-Do List App

A modern full-stack To-Do application that allows users to manage tasks efficiently using a clean and simple interface.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ✏️ Edit existing tasks
- ❌ Delete tasks
- 🔍 Filter tasks by status and priority
- 📅 Set due date for tasks
- 💾 Persistent storage using database

---

## 🛠️ Tech Stack

### 🔹 Backend
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

### 🔹 Frontend
- React (Hooks)
- Fetch / Axios (API communication)
- CSS (modern UI styling)

---

## ▶️ How to Run

### 🔹 Backend (Spring Boot)

1. Open project in IntelliJ
2. Run the application
3. Server will start on:  http://localhost:8080

### 🔹 Frontend (React)

```bash
cd frontend
npm install
npm start   

```
